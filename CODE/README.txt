 INCLUDE
- 2 files: one for MASTER, one for SLAVE
- Communication via HC-05 Bluetooth Module

 HOW TO USE:
- Using Arduino IDE to open
- Upload 'master.ino' to MASTER, 'slave.ino' to SLAVE
- Wait the MASTER for 3 seconds to setup
- If the led RX,TX on both MASTER and SLAVE board start to blink --> Ready to use
